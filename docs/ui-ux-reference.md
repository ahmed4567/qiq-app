# UI/UX reference for QualityIQ

This project now uses the UI kits in `E:\work\UI-UX` as inspiration for the portal redesign.

## Chosen direction

- **Primary dashboard style**: `DashStack - Free Admin Dashboard UI Kit - Admin & Dashboard Ui Kit - Admin Dashboard (Community).zip`
- **Navigation pattern**: `Sidebar Navigation Menu for Dashboard – Animation (Dark & Light Versions) (Community).zip`

## What to carry forward

- Dense but readable sidebar navigation
- Role switcher for admin, reviewer, and agent
- Card-based dashboard layout with strong spacing
- Dark admin theme with bright accent color
- Secondary module chips for future features

## Role views

- **Admin**: permissions, sync health, policy, and system controls
- **Reviewer**: queue, scoring workspace, and coaching actions
- **Agent**: personal reviewed cases, scores, and improvement history
